Used to create the structure of the website
It defines what appears on the page

👉 In your code:

<header> → navigation bar
<section> → different parts (Hero, Departments, About, Contact)
<form> → contact form
 2. CSS (Cascading Style Sheets)
Used for design and styling
Controls colors, layout, spacing, and responsiveness

👉 In your code:

Background colors
Font sizes
Layout using Flexbox
Responsive design
 2. Tools / Concepts Used
 1. Flexbox
A CSS layout system to align items

Example:

display:flex;
justify-content:space-between;
align-items:center;

📌 Used for:

Navbar alignment
Department cards layout
2. Responsive Design
Makes the website look good on mobile devices
@media(max-width:768px)

📌 Used for:

Changing layout on smaller screens
Adjusting text size
3. Background Image
background:url(...) center/cover;

📌 Used in:

Hero section (full-screen image)
4. Box Shadow & Border Radius
box-shadow:0 0 10px rgba(0,0,0,0.2);
border-radius:10px;

📌 Used for:

Creating modern card design
